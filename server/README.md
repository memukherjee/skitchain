## 📦 Setup


## 📁 Folder Structure

- `controllers/` – Route handlers
- `routes/` – Route definitions
- `middlewares/` – Custom Express middleware
- `models/` – Data models or types
- `utils/` – Utility/helper functions

## ✅ Features

- Express.js
- CORS
- dotenv support
- TypeScript
